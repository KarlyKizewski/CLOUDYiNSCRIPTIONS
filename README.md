# Cloudy Inscriptions - First Personal Blog <br />
https://cloudyinscriptions.herokuapp.com/<br />
First Personal Project! 
A Rails blog to code about Rails.

Instead of simply implementing gem simple_form, I opted for basic HTML form_for. It was nice to get a feel for this alternative.<br />
